# 价格专员业务知识库

生成时间：2026-07-20T14:06:27.438869

本知识库由项目目录中的历史 Excel、控价标准和店铺档案生成。原始文件未修改。

## 知识域

- `store_master.csv`：网络店铺档案主表。
- `drug_master.csv`：药品/商品名/通用名/分类及历史覆盖。
- `drug_package_master.csv`：药品与标准化规格、包装数量和最小单位。
- `control_price_rules.csv`：控价规则，当前判断以 `价格标准表.md` 为准。
- `responsibility_relations.csv`：店铺、责任单位、责任人及监控层级。
- `monitor_task_master.csv`：由历史链接聚合出的复查任务；只有商品身份完整且至少跨 2 个采集日期出现的链接默认启用。
- `price_observations_clean.csv`：统一后的 1–6 月历史价格事实，保留来源侧控价和低价标记，并计算统一单件价。
- `data_quality_issues.csv`：未识别药品、店铺未匹配、重复、公式不一致和缺少商品身份等问题。

## 质量摘要

- 价格事实：185,529 条（识别后原始行 186,619，已移除完全重复 1,090）；监控任务：39,399 条；店铺：10,507 家。
- 未识别药品行：354；店铺未匹配行：6,075。
- 趣维单盒价公式不一致：164 行；这些行标记为隔离，不参与可信计算。
- 规则解释：历史来源中的“是否低价/是否破价”仅作历史事实；当前规则判断使用标准表控价，并要求规格、包装和最小单位可解析。
- 安托工作簿的 Excel 维度元数据异常，读取时已按实际单元格重置维度；源文件未改写。

## 使用边界

正式价格结论必须经过详情页确认；搜索列表价只能作为候选发现。责任关系以当前店铺档案为准；历史中无法匹配责任关系的任务保持观察态，不自动通知。
