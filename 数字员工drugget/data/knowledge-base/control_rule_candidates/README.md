# Stage 2A 指导价候选与业务确认边界

`stage2a_5_drugs.csv` 是业务确认模板，不是正式控价规则输入。它不能直接
导入 `control_price_rules.csv`，更不能同步数据库。模板中的指导价均来自有效的
`价格标准表.md`；`business_confirmed` 仅记录额外人工确认，不能作为指定表指导价
的运行时资格门槛。所有未确认行的确认人、确认时间和审批凭证均保持空白。

五个代表药的选择如下：

| 角色 | 药品 / 规格 | 选择依据 | 当前可比较性 |
| --- | --- | --- | --- |
| 已确认基线 | 葛泰 地奥司明片 `0.45g*20片` | 唯一已有业务确认、完整规格且真实详情页 dry-run 的样本；同时有 20/24 片多规格和多盒总价风险。 | 指导价已匹配；页面证据完整时可比较 |
| 单 SKU | 晴诺舒 米拉贝隆缓释片 `50mg*20片` | `drug_package_master.csv` 只有一个历史规格候选，适合作为普通单 SKU replay。 | 指导价已匹配；待页面 SKU 证据 |
| 多 SKU | 优立维 硫酸氢氯吡格雷片 `75mg*36片` | 有 `75mg*36片`、`75mg*48片` 两个历史规格候选，须验证 SKU 切换后的价格归属。 | 指导价已匹配；待页面 SKU 证据 |
| 复杂包装 | 安多林 阿瑞匹坦胶囊 `80mg*2粒+125mg*1粒` | 组合包装需要验证合计粒数、规格和单粒换算，不能按普通单强度商品处理。 | 指导价已匹配；待页面包装证据 |
| 阶梯/起购风险 | 畅凡 芦比前列酮软胶囊 `24μg*10粒` | 有 10/14/16/28 粒多个包装候选，适合验证多盒价、起购量与阶梯报价。 | 指导价已匹配；待页面报价证据 |

原始资料未提供生产厂家字段，故模板中 `manufacturer` 保持空白，待业务在
审批时按可追溯资料补全；不以历史包规或品牌名推断厂家。

可用以下只读命令生成全量 30 品种缺口 JSON：

```bash
uv run python scripts/stage2a_control_price_gap_report.py --today 2026-07-23 \
  > outputs/stage2a/control_price_gap_report.json
```
